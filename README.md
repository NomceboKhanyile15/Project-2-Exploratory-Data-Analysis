Project 2: Exploratory Data Analysis (EDA)

Overview

This project was completed as part of the DecodeLabs Data Analytics Internship Programme.

The objective of the project was to perform Exploratory Data Analysis (EDA) on a cleaned dataset to identify patterns, trends, distributions, and potential outliers.

Tools Used

- Microsoft Excel
- GitHub

Analysis Performed

- Descriptive Statistics
- Product Distribution Analysis
- Payment Method Analysis
- Order Status Analysis
- Referral Source Analysis
- Coupon Code Analysis
- Trend Analysis
- Outlier Detection using Box & Whisker Plot

Key Findings

- Printers were the most frequently purchased product.
- Online payments were the most commonly used payment method.
- Instagram generated the highest number of referrals.
- Cancelled orders slightly exceeded delivered orders.
- Average order values declined between 2023 and 2025.
- Potential outliers were identified within the Total Price variable.
